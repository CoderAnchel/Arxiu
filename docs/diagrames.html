1 · Model de dades

Esquema relacional MySQL 8.0. Soft delete a totes les entitats acadèmiques (camp deleted_at) — l'arxiu és permanent: cap registre no s'elimina mai, només es marca com a no actiu. Les notes per RA i la nota final del mòdul viuen en taules separades perquè la nota final pot ser una mitjana automàtica ponderada o una sobreescriptura manual del docent.

2 · Validacions per crear un usuari

L'admin crea un docent des de la UI. El sistema genera una contrasenya forta i la mostra una sola vegada; el receptor la rep per email o manualment. Al primer accés és obligat canviar-la. Cada pas valida les invariants i deixa rastre a audit_logs.

3 · Estats d'una avaluació

Cada avaluació passa per quatre fases. Només l'admin pot avançar o desfer transicions, i només a estats adjacents — mai es salten nivells. Cada estat habilita un conjunt de permisos diferents per a docents, tutors i admin. Totes les transicions queden a audit_logs amb actor, abans i després. 

stateDiagram-v2
    [*] --> oberta : Admin crea avaluació
POST /avaluacions

    oberta : oberta — PERÍODE DE CONFIGURACIÓ\n✓ Admin edita estructura\n✗ Cap docent introdueix notes\n✗ Tutor només lectura
    docent : docent — INTRODUCCIÓ DE NOTES\n✓ Profe amb assignacio_docent edita RA\n✗ Tutor encara no edita\n✓ Admin pot editar tot
    junta  : junta — JUNTA D'AVALUACIÓ\n✓ Tutor edita RA + nota final manual\n✓ Admin pot editar tot\n✗ Profes sense tutoria, lectura
    tancada: tancada — TANCADA\n✗ Ningú edita notes\n✓ Generació butlletins PDF\n✓ Enviament d'emails\n✓ Arxivat permanent

    oberta --> docent : admin avança\nPOST /avaluacions/:id/transition\n{target: docent}
    docent --> junta : admin avança\n{target: junta}
    junta --> tancada : admin avança\n{target: tancada}\nset data_tancament

    docent --> oberta : admin rollback\n{target: oberta}\ncorrecció estructura
    junta --> docent : admin rollback\n{target: docent}\nmés notes per profes
    tancada --> junta : admin rollback\n{target: junta}\nclear data_tancament

    tancada --> [*] : Història preservada\n(arxiu permanent)
